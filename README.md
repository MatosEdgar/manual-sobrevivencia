# Manual de Sobrevivência: Apocalipse Zumbi

Este é o meu guia definitivo para sobreviver em um mundo infestado.
Criado dia a dia enquanto meus estudos de **Git e Github**

## Regra Número 1:
Sempre saiba onde estão as suas ferramentas.

## lista de Suprimentos Essenciais (Mochila)
* 1 Litro de água (Renovável)
* Enlatado (comida não estraga rápido)
* Lanterna de dínamo (sem pilhas)
* Kit de primeiros socorros 

## Tarefas Diárias para Sobrevivência
1. Verificar o perímetro do abrigo.
2. Coletar água limpa.
3. Rastrear sinais de rádio.
